Interactive Elements

Navigation Button

CSS selector:

.button-nav

Sample code:

<a href="#" class="button-nav">Contact Us</a>

Rendered element:

Contact Us

In-text link

CSS Selector:

.intext-link

Sample code:

<a href="#" class="intext-link">Sample Link</a>

Rendered element:

Sample Link

Submit/Reset Buttons

CSS selector:

.button-submit

.button-reset

Sample code:

<input class="button-submit" type="submit"> <input class="button-reset" type="reset">

Rendered element:

Text Input Field

.sample-label

.text-field

Bulleted Lists

CSS selector:

.bullet-list

Numbered Lists

.number-list

  3. Item Three
